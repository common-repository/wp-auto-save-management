Manage Your Auto Save And Post Revisions From The Admin Interface

== Description ==

Manage Your Auto Save And Post Revisions From The Admin Interface.

Under Settings -> General There are 3 new options to manage this WordPress functionality.


== Installation ==

1. Upload The wp-auto-save-management folder to the `/wp-content/plugins/` directory on your server
2. Activate the plugin in the admin interface
3. Thats All!
